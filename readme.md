## Usage

Returns the current working directory in string format.

```rust
let path = match where_i_am()?;

println!("{}", path);
```

## CLI

### Installation

```bash
cargo install whereiam
```

Or use a binary from
[releases](https://github.com/UltiRequiem/whereiam/releases/latest).
