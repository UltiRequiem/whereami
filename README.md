## Usage

Just like [`pwd`](https://en.wikipedia.org/wiki/Pwd):

```bash
whereiam
```

> E.g: `/home/justice/repos/whereiam`

### Installation

```bash
cargo install whereiam
```

Or use a binary from [releases](https://github.com/UltiRequiem/whereiam/releases/latest).
